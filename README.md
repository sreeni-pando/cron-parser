# Cron-Parser
Cron parser as a separate library function

## Run Locally

Install packages

```bash
  npm install
```
Run the script to test the O/P
test-cron.js is the name of file where we call cron-lib class function

```bash
  node test-cron.js  "30 9 15 * * /usr/"
```
